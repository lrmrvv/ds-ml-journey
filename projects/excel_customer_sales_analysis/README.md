# Excel Customer Sales Analysis

## Описание проекта
Анализ данных продаж с целью выявления ключевых факторов,
влияющих на выручку, поведение клиентов и эффективность каналов продаж.

## Бизнес-задачи
- Определить регионы с наибольшей выручкой
- Проанализировать эффективность каналов продаж
- Выявить наиболее и наименее продаваемые категории товаров
- Проверить наличие сезонности продаж

## Используемые инструменты
- Excel 
- Базовый анализ данных
